## Visit GitHub.com

These instructions are subject to change. They were accurate as of July 2023.

To create a GitHub account, visit <https://GitHub.com/>; select 
"<span style="color:red; font-weight:bold; font-size:larger;">Sign Up</span>." 
If you already have a GitHub account, select 
"<span style="color:red; font-weight:bold; font-size:larger;">Sign In</span>."
