## Visit GitHub.com

These instructions are subject to change. They were accurate as of July 2023.

To create a GitHub account, visit <https://GitHub.com/>; select 
"<span style="color:red; font-weight:bold; font-size:larger;">Sign Up</span>." 

If you already have a GitHub account, select 
"<span style="color:red; font-weight:bold; font-size:larger;">Sign In</span>."

## UserName

* GitHub UserName (account) are not 
  <span style="color:red; font-weight:bold; font-size:larger;">case-sensitive</span>.
  Unfortunately, Linux accounts are!
  Preferably, use all lowercase characters for your GitHub UserName account.
* In this example, the user account is "cbkadal."
  * Remember: <span style="color:red; font-weight:bold; font-size:larger;">you are not cbkadal!</span>
